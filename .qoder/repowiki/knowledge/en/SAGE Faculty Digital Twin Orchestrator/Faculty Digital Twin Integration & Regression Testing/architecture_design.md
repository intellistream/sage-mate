- **Test Infrastructure**: Uses `pytest` with a centralized `conftest.py` that bootstraps the runtime environment, manages `sys.path` for sibling repositories (SAGE, sageVDB), and enforces offline mode for HuggingFace models.
- **Isolation Strategy**: Heavily relies on dependency injection and monkeypatching (via `isolated_availability_store` and custom `LLMClient` stubs) to decouple tests from external LLM APIs and persistent storage, ensuring deterministic execution.
- **Coverage Domains**: Organized into functional clusters: agentic workflow logic (`test_agentic_workflow.py`), real-time communication via SSE (`test_workflow_sse_keepalive.py`), extensibility via capability plugins (`test_capability_plugins.py`), and administrative operations like booking and auth (`test_admin_auth.py`, `test_meeting_service.py`).
- **Evidence**: Key architectural contracts are verified in `test_chat_post_answer_background.py` (async background task splitting) and `test_prompt_soft_cap.py` (context window management).
