- **Runtime Bootstrap**: `conftest.py` orchestrates a isolated test environment by manipulating `sys.path` to include sibling repositories (`SAGE`, `sageVDB`) and enforcing offline mode for HuggingFace models to prevent network dependencies.
- **Workflow Trace Contract**: Tests like `test_chat_pipeline_dag.py` enforce a strict, canonical sequence of `workflow_trace` steps (e.g., `bootstrap` -> `interaction_understand` -> `llm_answer`), ensuring the DAG-based planner maintains backward compatibility with the legacy linear pipeline.
- **Agentic Simulation**: Complex scenarios in `test_agentic_workflow.py` use specialized mock LLM clients (e.g., `IntentAwareLLMClient`, `ShadowPlanningLLMClient`) to simulate intent classification, booking flows, and shadow-planner comparisons without external API calls.
- **Data-Driven Evaluation**: Identity and workflow correctness are verified via parameterized tests (`test_identity_eval.py`, `test_workflow_eval.py`) that load structured JSON scenarios from `tests/data/`, decoupling test logic from specific input cases.
- **Frontend Contract Guardrails**: `test_frontend_contract.py` performs static analysis on `app.js` and `index.html` to ensure DOM IDs used in API payloads exist in the UI, preventing 422 errors from mismatched frontend-backend interfaces.
